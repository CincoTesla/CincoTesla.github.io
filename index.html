Descripción del Proyecto:

Objetivos del Proyecto:

  • Comprender el funcionamiento de los sistemas de consumo eléctrico para desarrollar una web de cálculos monetarios y energéticos.
  • Visualizar e implementar interfaces amigables para mejorar la experiencia del usuario.
  • Utilizar herramientas innovadoras, como Visual Studio Code, Python, JavaScript, HTML, CSS, y el framework Django.

Información del proyecto:

  • Con nuestro Proyecto de Gestión de Sistema Eléctrico, buscamos consolidar una página web que permita al usuario seleccionar electrodomésticos para calcular su consumo energético y monetario en el hogar. Presentaremos los resultados mediante estadísticas y gráficas interactivas.
